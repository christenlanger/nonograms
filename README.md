# Nonogram Site

A pet project currently being developed. Everything is still in the conceptual phase.

## Current progress

The composer, board presentation, and solving logic is in place. Once the core components are in place, proper pages will be created to present the site.

- The user can solve pre-made puzzles loaded from a JSON file
- The user can create custom puzzles and store them locally
- The user can solve those puzzles as well.

## Goal of this project

The ultimate goal is to have a website that allows users to log in, create their own nonograms, and share them to other users.
